mod_pubsub Tree Widgets
  From the mod_pubsub demos  
   


tree1, tree2, and tree3 are three different versions of a frameset-embeddable, topic-choosing tree widget.

tree1 uses Unicode graphics characters. tree_demo.html is a demo application using tree1, which uses the tree widget in the left frame to choose the initial topic for the introspect application, which is loaded in the right frame.

tree2 uses images from Joust (www.ivanpeters.com) to get a Windows or OS/2 look. To use this widget, you'll need to download those separately.

tree3 uses ASCII images. It is the most portable and the ugliest of the three widgets.
