Overview

The KN FlashGraph component is a Flash Movie that allows you to use JavaScript to dynamically control the position and size of a bar graph. Updates to the graph take place in realtime. To see an example of how this works with mod_pubsub, see the example in examples/kn_flashgraph.

Installing KN FlashGraph

To install KN FlashGraph:

  • Include the following object code segment in your application. Make sure that you put your application in the same folder as kn_bargraph.swf.
<OBJECT classid="clsid:D27CDB6E-AE6D-11cf-96B8-444553540000"
codebase="http://active.macromedia.com/flash2/cabs/swflash.cab#version=5,0,41,0"
ID=movie1 WIDTH=520 HEIGHT=340>
<PARAM NAME=movie VALUE="kn_bargraph.swf">
<PARAM NAME=quality VALUE=high>
<PARAM NAME=bgcolor VALUE=#FFFFFF> 
<EMBED NAME="movie1" src="kn_bargraph.swf" SWLiveConnect="true" quality=high bgcolor=#FFFFFF
WIDTH=520 HEIGHT=340 TYPE="application/x-shockwave-flash" 
PLUGINSPAGE=
"http://www.macromedia.com/shockwave/download/index.cgi?P1_Prod_Version=ShockwaveFlash">
</EMBED>
</OBJECT>

Axis Properties

The following table provides a list of x-axis properties.

Axis Properties Description
X Axis Label The text label to put on the x-axis.
X Axis Length Length of the axis in pixels.
X Axis Min The minimum coordinate on the x-axis.
X Axis Max The maximum coordinate on the x-axis.
X Axis SubDivisions The number of sections that the x-axis is divided into.

The following table provides a list of y-axis properties.

Axis Properties Description
Y Axis Label The text label to put on the y-axis.
Y Axis Length Length of the axis in pixels.
Y Axis Min The minimum coordinate on the y-axis.
Y Axis Max The maximum coordinate on the y-axis.
Y Axis SubDivisions The number of sections that the y-axis is divided into.

Bar Properties

The following table provides a list of bar properties.

Properties Description
Bar Label The text label to put on the bar.
Bar X The X coordinate of the bar from the left edge.
Bar Y The height of the bar.
Bar Width The width of the bar in pixels.
Bar Color Hex color of the bar. For example: "FFCCOO"
Bar State Specifies whether or not to use a frame in the bar symbol.

1 = labeled bars
2 = unlabeled bars

Resizing the FlashGraph Movie

To change the size of the flash movie:

  1. Edit the FLA included in the source folder in your jscomponents download.
  2. Export the re-sized movie.
  3. Change the width and height attributes in both the object and embed tags to match the new size of the graphic or movie.

Using JavaScript to Control the Flash Movie

The constructors for the build_x_scale, build_y_scale, and build_bar functions are provided below. Examples of how to implement each function are also provided. In the implementation examples, use a "null" string to pass a null parameter.

Building the X and Y Axis

You can position a bar to match the x-axis coordinate system or side-by-side another bar.

  • To position a bar to match the x axis coordinate system, specify the min and max values of the x-axis. When the min and max values have been specified, the x and width values of the build_bar command match the x-axis coordinate system and the bar is placed accordingly.
  • To position a bar side-by-side another bar, use the subDivisions parameter to create a slot for each bar. When the subDivision parameter is specified, the x value of the build_bar command refers to the index of the slot you want to use. The index value begins at zero.

build_x_scale (label, length, min, max, subDivisions)

Implementation examples:

  • flash_command("build_x_scale", "x-axis", 300, 0, 100, 200)
  • flash_command("build_x_scale", "x-axis", 300, "null", "null", 100, 200)

build_y_scale (label, length, min, max, subDivisions)

Implementation examples:

  • flash_command("build_y_scale", "y-axis", 300, 0, 100, 200)
  • flash_command("build_y_scale", "y-axis", 300, "null", "null", 100, 200)

Building Bars

build_bar (label, x, y, width, color, state)

Implementation examples:

  • flash_command("build_bar", "bar1", 0, 50, 10, "FFCCOO", 2)
  • flash_command("build_bar", "bar1", 0, 50, "null", "FFCCOO", 2)
