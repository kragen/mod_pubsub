KN_HTML Token Example

This example demonstrates the use of KN_HTML tokens to insert KN event property values into HTML.

Enter text into the Payload text box and click the button to publish the event. The grey box contains a line of HTML with the KN tokens {{kn_id}} and {{kn_payload}} inserted.

Payload:
The payload of Event #{{kn_id}} is "{{kn_payload}}".
