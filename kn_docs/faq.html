mod_pubsub

Asynchronous publish and subscribe messaging module for the Apache web server...

What is mod_pubsub?

mod_pubsub is a set of libraries and scripts that enable asynchronous publish and subscribe messaging over HTTP. mod_pubsub extends Apache by running within its mod_perl Web Server module.

What's the benefit of developing with mod_pubsub?

Real-time data delivery to & from Web Browsers without refreshing; without installing client-side software; and without Applets, ActiveX, or Plug-ins. This is useful for live portals & dashboards and Web Browser notifications.

What about other Web Servers? What about if I have no Web Server?

mod_pubsub's scripts can be used with other Web Servers as well. We also provide a separate, feature-compatible standalone PubSub Server written in Python that can be run from the command-line.

Are there any other products like mod_pubsub?

One similar project is Pushlets. Unlike mod_pubsub, Pushlets can only maintain a few simultaneous connections, are not built to be run with Apache, and require Java programming. With mod_pubsub, if you know how to use HTTP, you can do PubSub. Courtesy of KnowNow.

What programming languages and platforms does mod_pubsub support?

Besides a JavaScript library that runs within a Browser's Web page, we also offer embeddable libraries to connect applications written in C; Visual Basic, Visual C++, and C# using ActiveX, COM, .NET, or PocketPC; Python, Perl, PHP, and Ruby; and Java. These libraries allow asynchronous publish and subscribe messaging over HTTP. Our goal is to connect to as many platforms as possible.

Cool, how can I try mod_pubsub?

Download the mod_pubsub tarball from our SourceForge project page. Also, join our mailing list to keep up to date on the development of mod_pubsub.
