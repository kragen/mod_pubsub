kn_xml.js Helper Library

Motivation

Convenience library included to add in SOAP and XML-related helper libraries.

Usage Synopsis

  <script src="../kn_lib/kn_xml.js"></script>

This library has occasionally caused problems in Netscape.

Currently, this includes the following JavaScript libraries:

  • xmljs/xml.js
  • object2xml.js
  • xml2object.js
  • soap2object.js
  • object2soap.js
  • kn_xml_msg.js
