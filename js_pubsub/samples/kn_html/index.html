KN HTML

This example demonstrates the use of PubSub HTML to insert event property values into HTML.

Enter text into the Payload text box and click the button to publish the event. The grey box contains a line of HTML with the PubSub tokens {{kn_id}} and {{kn_payload}} inserted.

Payload:
The payload of Event #{{kn_id}} is "{{kn_payload}}".
