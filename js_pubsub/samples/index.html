PubSub JavaScript Samples and Components

Here are the sample implementations of some end-user applications and components. All of these applications work in real time through browsers; to implement them, you embed code in HTML pages.

  • Sample Applications
    • The Buddy List is a personal list of one's "buddies." This list is used in conjunction with the Instant Messaging application.
    • The Chatlet is an embedded Chatlet window that any number of users can access to chat with each other.
    • Chat with Presence is an embedded Chatlet window that any number of users can access to chat with each other and watch other users enter and leave the chatroom.
    • The Instant Messaging Client makes it possible for two people to share a private chat. They can use their Buddy List to manage and contact their friends.
    • Screen Pops can be used to broadcast on-the-spot messages to one or more users.
    • The ScreenPrompts make it possible to send alerts to individual users.
  • Demonstrations of Component Applications
    • The ActivePanel sample is a demonstration of the ActivePanel component, which can be used to create embedded, "live" content areas in static Web pages without resorting to frames. The ActivePanel uses PubSub technology to make it possible to update these content areas in real time.
    • The PubSub HTML sample demonstrates the capability of sending data in real time to a location in a Web page without having to use frames or the Refresh command.
    • The JavaScript Lightweight Web Service Library can be used to support and create other applications.
